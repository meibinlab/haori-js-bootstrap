# Changelog

All notable changes to this project will be documented in this file.

## 0.2.0 - 2026-04-21

- Renamed the npm package from haori-js-bootstrap to haori-bootstrap.
- Renamed the distributed bundle files to haori-bootstrap.js and haori-bootstrap.iife.js.
- Updated README, Japanese README, demo pages, CDN references, and release instructions for the new package name.
- Added migration guidance so the previous package name can be deprecated on npm.

## 0.1.0 - 2026-04-21

- First public release of Haori.js Bootstrap.
- Added npm release automation triggered by GitHub Release publication.
- Added release asset automation that uploads dist.zip built from dist/.
- Documented npm installation, CDN usage, and release steps.
